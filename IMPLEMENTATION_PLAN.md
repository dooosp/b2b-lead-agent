# B2B 리드 에이전트 — 콘텐츠 전문성 강화

> 작성: 2026-02-14
> 대상: `worker/index.js` (3172줄) — 텍스트/라벨만 변경, 로직 무변경

## 문제
기능적으로는 완성이지만 UI 텍스트가 콩글리시 + 기술 용어 노출 + 캐주얼한 표현 → B2B 영업 전문 도구 수준으로 격상

---

## 1. 콩글리시 → 전문 한국어 (7건)

| 현재 | 변경 | 위치 |
|------|------|------|
| 컨택완료 | 접촉 완료 | statusLabels 2곳 |
| 팔로업 (전체) | 후속 조치 | 대시보드, 상세페이지, 칸반 (~6곳) |
| 딜 금액 | 예상 계약액 | 상세페이지 label |
| 멘트 | 메시지 | 롤플레이 input placeholder |
| 페인포인트 | 고객 과제 | 상세페이지 enrichment 라벨 |
| 액션 아이템 | 후속 실행 항목 | 상세페이지 enrichment 라벨 |
| 파이프라인 가치 | 진행 중 거래 총액 | 대시보드 메트릭 2곳 |

## 2. 한영 혼용 → 통일 (2건)

| 현재 | 변경 | 위치 |
|------|------|------|
| 영업 Pitch | 영업 제안 | 리드카드, 상세페이지, 셀프서비스 결과 (~4곳) |
| Grade A | A등급 | 대시보드 메트릭 카드 1곳 |

## 3. 기술 노출 제거 (5건)

| 현재 | 변경 | 위치 |
|------|------|------|
| Gemini AI가 제안서를 작성하고 있습니다 | AI가 제안서를 작성하고 있습니다 | PPT 페이지 |
| 미분석 리드를 Gemini AI로 심층 분석합니다 | 미분석 리드를 AI로 심층 분석합니다 | 일괄 enrichment 바 |
| AI가 프로필을 생성하고 뉴스를 분석하고 있습니다 | 프로필 생성 및 뉴스 분석 중입니다 | 셀프서비스 로딩 |
| 산업 뉴스 수집 → Gemini AI 분석 → 리드 리포트 이메일 발송 | 뉴스 기반 영업 기회 분석 후 리포트를 발송합니다 | 메인페이지 info |
| 이미 분석된 리드입니다. ?force=true로 재실행하세요 | 이미 분석된 리드입니다. 재분석 버튼을 이용하세요 | API 에러 |

## 4. 브랜딩 격상 (3건)

| 현재 | 변경 | 위치 |
|------|------|------|
| B2B 리드 에이전트 | B2B Sales Intelligence | title, h1, PWA manifest name |
| AI 기반 B2B 영업 기회 발굴 | AI 기반 영업 인텔리전스 플랫폼 | subtitle, PWA description |
| 영업 시뮬레이터 | 영업 역량 시뮬레이션 | 롤플레이 h1, 메인 nav 버튼 |

## 5. AI 프롬프트 출력 품질 강화 (enrichment)

enrichment 프롬프트 JSON 키 설명을 더 구체적으로:

| 키 | 현재 설명 | 변경 |
|-----|---------|------|
| painPoints | "페인포인트1: 설명" | "비용/규제/경쟁/기술 관점의 구체적 과제 (정량 수치 포함)" |
| actionItems | "액션1: 구체적 다음 단계" | "1주 내 실행 가능한 후속 조치 (담당 부서/직급 포함)" |
| salesPitch | "고객 페인포인트 직접 언급, 구체적 솔루션 제안" | "고객 과제를 먼저 언급하고, 정량적 해결 방안과 레퍼런스를 포함한 3문장 영업 제안" |
